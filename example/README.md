# flow_widget example

Polished demo app covering:

- Weather (timeline)
- Habit tracker (interactive)
- Calendar
- Fitness
- Music player
- Finance dashboard
- News (remote images)
- Photo
- Countdown (scheduled timeline)
- Checklist (multiple instances + pin)
- Live Activities / Dynamic Island

```bash
flutter run
```

Configure an App Group id in `lib/main.dart` and run the CLI to scaffold native widget targets:

```bash
dart run flow_widget_cli:flow_widget create --name WeatherWidget --platforms android,ios
```
